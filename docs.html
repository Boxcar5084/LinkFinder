What is LinkFinder?

LinkFinder is a forensic investigation tool that discovers transaction paths between sets of Bitcoin addresses. Given a list of source addresses and target addresses, it traces through blockchain transactions to find how they are connected.

Using a bidirectional breadth-first search algorithm, LinkFinder efficiently finds the shortest paths connecting addresses while intelligently filtering out noise like mixing services, exchange wallets, and airdrop transactions.

Usage Guide

Open the Web Interface

After starting both the backend and frontend, navigate to http://localhost:8501 in your browser. The Streamlit UI should open automatically.

Enter Your Addresses

Go to the "New Trace" tab. Enter your source addresses in List A (one per line) and your target addresses in List B (one per line).

Field Description
List A Source Bitcoin addresses you want to trace FROM
List B Target Bitcoin addresses you want to trace TO
Max Depth Maximum transaction hops to search (default: 5)
Block Range Optional: Limit search to specific block heights

Start the Trace

Click "Start Trace" to begin. Switch to the "Active Sessions" tab to monitor progress. You'll see real-time updates showing addresses examined and connections found.

View Results

Connections appear in the "Active Sessions" tab. Switch to the "Found Connections" tab for a complete list of all discovered paths, or check the exports/ folder.

Pause & Resume

For long traces, you can click "Stop" to save a checkpoint. Later, go to the "Checkpoints" tab and click "Resume Latest" to continue where you left off.

Configuration

Use the "Settings" tab to switch between API providers (Mempool, ElectrumX, Blockchain.info), set API keys, and adjust transaction filtering thresholds.
